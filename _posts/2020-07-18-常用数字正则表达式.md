## 整数

```text
^-?[1-9]\d*|0$
```

### 正整数
```text
^[1-9]\d*$
```
### 负整数
```text
^-[1-9]\d*$
```
### 自然数/非负整数
```text
^[1-9]\d*|0$
```
### 非正整数
```text
^-[1-9]\d*|0$
```
## 小数/浮点数
```text
^-?([1-9]\d*|0)\.\d+$
```
### 正小数/正浮点数
```text
^([1-9]\d*\.\d+|0\.\d*[1-9]\d*)$
```
### 负小数/负浮点数
```text
^-([1-9]\d*\.\d+|0\.\d*[1-9]\d*)$
```
### 非负小数/非负浮点数
```text
^([1-9]\d*|0)\.\d+$
```
### 非负小数/非负浮点数
```text
^-([1-9]\d*|0)\.\d+$
```
## 实数
```text
^-?([1-9]\d*|0)(\.\d+)?$
```
### 正实数
```text
^([1-9]\d*(\.\d+)?|0\.\d*[1-9]\d*)$
```
### 负实数
```text
^-([1-9]\d*(\.\d+)?|0\.\d*[1-9]\d*)$
```
### 非负实数
```txt
^([1-9]\d*|0)(\.\d+)?$
```
### 非正实数
```text
^-([1-9]\d*|0)(\.\d+)?$
```
